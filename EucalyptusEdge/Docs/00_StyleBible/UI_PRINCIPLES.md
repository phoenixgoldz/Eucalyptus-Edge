# EUCALYPTUS EDGE — UI PRINCIPLES
**v1.0 · Authored by Phoenix (PhoenixGold Game Studios)**
Read this before the Style Guide. The Style Guide says *how*. This says *why*.

1. Gameplay always comes first.

2. Menus should feel handcrafted.

3. Decorative detail increases with importance.
   HUD < Menu < Character Select < Victory

4. Every interactive object responds.

5. Animation should support clarity.

6. Magic is subtle.

7. Every crystal emits Edge Energy.

8. Violet always means Blight.

9. Gold represents honor.

10. The UI should feel alive, never distracting.

---
*If an implementation decision isn't covered by the Style Guide, resolve it against these ten lines. If it violates one of them, it's wrong — no matter how good it looks.*
